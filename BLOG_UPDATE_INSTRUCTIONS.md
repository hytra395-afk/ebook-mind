## 🚀 Qtradicional LÀM VIỆC

### Bước 1: Bạn Cung Cấp Nội Dung
Gửi cho tôi nội dung bài blog với format:
```
Tiêu đề: [Tiêu đề bài viết]
----------
[Nội dung bài viết - có thể dài, có chia đoạn, có ví dụ, case study...]
```

### Bước 2: Tôi Format & Tạo SQL
- Format HTML đầy đủ (H2, H3, ul, ol, blockquote, p, strong, em, a)
- Tạo excerpt hấp dẫn (150-200 ký tự)
- Tạo meta_title & meta_description (SEO-optimized)
- Tạo keywords array
- Tính read_time
- Tạo SQL UPDATE script

### Bước 3: Bạn Chạy SQL Trong Supabase
Copy SQL script → Paste vào Supabase SQL Editor → Click RUN
